## 🧠 Our Philosophy

**Human-gated AI.** Models are specialized tools in a deterministic pipeline, not autonomous agents. We believe in:
- 📝 **Spec first** — Define what before how
- 🏗️ **Architecture second** — Design with intention
- 🛠️ **Implementation last** — Code only after approval
- 🚪 **Human gates** — Review at every decision point
